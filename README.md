## Iteration 5
星巴克数据分析工具 需求5

- 显示店铺信息
    + 用户点击查询到的店铺，弹出对话框显示店铺信息（包括店铺基本信息和评价信息）
    
- 对店铺进行评分
    + 用户可以对所选择的店铺进行评分，如果评分分数不合法，则提示用户
    
- 计算店铺平均评分
    + 若某店铺被多次输入评分，将所有评分进行记录，并计算其平均分
    
- 特殊标明高分店铺
    + 若某店铺平均评分高于8分，则在top-k、range、top-k+关键词的查询结果里用特殊记号标明。
